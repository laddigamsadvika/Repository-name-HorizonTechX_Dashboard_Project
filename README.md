# 📊✨ Academic Performance Dashboard | Data Visualization Project

## 🌟 Project Overview
This project focuses on transforming raw student academic performance data into a **professional dashboard** for visual storytelling, trend analysis, and data-driven decision-making 📈.

The dashboard was designed to present key insights in a clear and impactful format using **Python and Matplotlib**.

🚀 Completed as part of the **Horizon TechX Artificial Intelligence & Data Science Internship Program**


## 🎯 Objectives
✅ Convert raw data into meaningful charts  
✅ Build a professional dashboard view  
✅ Support decision-making with insights  
✅ Present trends and patterns visually  
✅ Improve data storytelling skills  

## 🛠️ Tools & Technologies
💻 Python  
📓 Jupyter Notebook  
🐼 Pandas  
📊 Matplotlib  

## 📂 Dataset Used
The project uses the **Students Performance Dataset**, which includes:

- 👩 Gender
- 🌍 Race/Ethnicity
- 🎓 Parental Education
- 🍱 Lunch Type
- 📝 Test Preparation Course
- ➕ Math Score
- 📖 Reading Score
- ✍️ Writing Score

## 📊 Dashboard Components

### 📈 1. Subject-wise Average Scores
Displays average performance in:
- Math
- Reading
- Writing

### 👩‍🎓 2. Gender Performance Comparison
Visual comparison of male and female student performance across subjects.

### 📝 3. Test Preparation Impact
Shows how completing the test preparation course affects academic scores.

### 🔗 4. Score Correlation
Scatter plot representing the relationship between:
- Reading Score
- Writing Score

## 💡 Key Insights
✨ Reading and writing scores are consistently higher than math  
✨ Female students perform better in language-related subjects  
✨ Test preparation significantly improves student performance  
✨ Strong positive correlation between reading and writing scores
